- I am a pretty heavy user of Claude Code in my professional work, and for my software side-projects. I always review any code I ship, so I’m not purely vibe-coding.

- I also use Claude for some day-to-day problem solving, as I often find it more useful than search.

- I have experimented with using AI to help me edit blog posts, but I don’t do this anymore. I’ve gone back and added a note to any of my old posts which were edited using AI.

- Other than what is explicitly marked, nothing on my personal website has been created or edited using AI. I want this website to be my thoughts, as imperfect as I am.
